# PCemail
PCemail is a special email system.

## Setup
How to Setup:

1. Download this repository.
2. Execute command in console:
```text
> npm install
```
3. Execute second command in console:
```text
> node index
```
4. Visit the server (http://localhost:3000) or [click this link](http://localhost:3000/)
5. Enjoy.
